# ag-grid-vue3 方法 (Methods)

## 获取 GridApi

通过 `grid-ready` 事件获取 `params.api`，进而调用 GridApi 方法。

```vue
<template>
  <AgGridVue ref="gridRef" @grid-ready="onGridReady" />
</template>

<script setup>
function onGridReady(params) {
  const api = params.api
  // 使用 api 调用 Grid API 方法
}
</script>
```

## GridApi 常用方法

| 分类 | 方法名 | 说明 |
|------|--------|------|
| 编辑 | stopEditing | 停止编辑 |
| 编辑 | startEditingCell | 开始编辑指定单元格 |
| 数据 | applyTransaction | 应用事务（add/remove/update） |
| 数据 | refreshClientSideRowModel | 刷新客户端行模型 |
| 行节点 | forEachNode | 遍历所有节点 |
| 行节点 | getRowNode | 根据 ID 获取行节点 |
| 列 | getColumnDefs | 获取列定义 |
| 列 | setColumnDefs | 设置列定义 |
| 列 | sizeColumnsToFit | 列宽自适应 |
| 筛选 | getFilterModel | 获取筛选模型 |
| 筛选 | setFilterModel | 设置筛选模型 |
| 选择 | getSelectedRows | 获取选中行 |
| 导出 | exportDataAsCsv | 导出 CSV |
| 刷新 | refreshCells | 刷新单元格 |

详细 API 参见 [ag-grid-community 方法](../ag-grid-community/方法.md)。
