# ag-grid-vue3 事件 (Events)

事件绑定方式：`@事件名="处理方法"`，事件名使用 **kebab-case**。

## 初始化

| 事件名 | 参数类型 | 说明 |
|--------|----------|------|
| grid-ready | GridReadyEvent | 表格初始化完成，可获取 api |

## 配件相关

| 事件名 | 参数类型 | 说明 |
|--------|----------|------|
| tool-panel-visible-changed | ToolPanelVisibleChangedEvent | 工具面板可见性变化 |
| context-menu-visible-changed | ContextMenuVisibleChangedEvent | 右键菜单可见性变化 |
| column-menu-visible-changed | ColumnMenuVisibleChangedEvent | 列菜单可见性变化 |

## 列相关

| 事件名 | 参数类型 | 说明 |
|--------|----------|------|
| column-visible | ColumnVisibleEvent | 列显示/隐藏 |
| column-pinned | ColumnPinnedEvent | 列固定/取消固定 |
| column-resized | ColumnResizedEvent | 列宽调整 |
| column-moved | ColumnMovedEvent | 列移动 |
| displayed-columns-changed | DisplayedColumnsChangedEvent | 展示列变化 |

## 编辑相关

| 事件名 | 参数类型 | 说明 |
|--------|----------|------|
| cell-value-changed | CellValueChangedEvent | 单元格值变化 |
| cell-editing-started | CellEditingStartedEvent | 单元格编辑开始 |
| cell-editing-stopped | CellEditingStoppedEvent | 单元格编辑结束 |
| row-editing-started | RowEditingStartedEvent | 行编辑开始 |
| row-editing-stopped | RowEditingStoppedEvent | 行编辑结束 |

## 选择相关

| 事件名 | 参数类型 | 说明 |
|--------|----------|------|
| selection-changed | SelectionChangedEvent | 选择变更 |
| row-selected | RowSelectedEvent | 行选中/取消选中 |
