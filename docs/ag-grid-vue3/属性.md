# ag-grid-vue3 属性 (Props)

## AgGridVue 组件

AgGridVue 接受 `GridOptions<TData>` 的全部属性，可直接作为 props 透传。Vue 中属性使用 **kebab-case**（如 `rowHeight` → `row-height`）。

## 核心属性

| 属性名 | 类型 | 说明 |
|--------|------|------|
| gridOptions | GridOptions\<TData\> | 一次性传入的完整配置对象 |
| modules | Module[] | AG Grid 模块（若使用模块化导入） |
| columnDefs | (ColDef \| ColGroupDef)[] | 列定义 |
| defaultColDef | ColDef | 默认列定义 |
| rowData | TData[] | 行数据 |
| context | any | 上下文，供自定义组件使用 |
| components | Record\<string, any\> | 自定义组件映射 |

## 常用属性

| 属性名 | 类型 | 说明 |
|--------|------|------|
| rowHeight | number | 行高 |
| headerHeight | number | 表头高度 |
| rowSelection | 'single' \| 'multiple' \| RowSelectionOptions | 行选择模式 |
| getRowId | GetRowIdFunc | 行 ID 获取函数 |
| statusBar | { statusPanels: StatusPanelDef[] } | 状态栏配置 |
| sideBar | SideBarDef \| string \| boolean | 侧边栏配置 |
| suppressContextMenu | boolean | 是否隐藏右键菜单 |
| loading | boolean | 是否显示加载覆盖层 |
| theme | 'legacy' \| Theme | 主题 |

## 编辑相关

| 属性名 | 类型 | 说明 |
|--------|------|------|
| singleClickEdit | boolean | 单击即编辑 |
| stopEditingWhenCellsLoseFocus | boolean | 失焦时停止编辑 |
| invalidEditValueMode | EditValidationCommitType | 校验失败时的行为 |
