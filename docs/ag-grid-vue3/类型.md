# ag-grid-vue3 类型 (Types)

## 导出的主要类型

| 类型名 | 说明 |
|--------|------|
| AgGridVue | Vue 3 表格组件 |

| 类型 | 说明 |

| ColDef | 列定义 |
| ColGroupDef | 列组定义 |
