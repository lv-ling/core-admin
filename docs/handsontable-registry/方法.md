# handsontable/registry 方法 (Methods)

## 导出的函数

| 方法名 | 参数 | 返回值 | 说明 |
|--------|------|--------|------|
| registerAllModules | () | void | 一次性注册所有 Handsontable 内置模块 |
| registerAllEditors | - | - | 注册所有内置编辑器 |
| registerAllRenderers | - | - | 注册所有内置渲染器 |
| registerAllValidators | - | - | 注册所有内置校验器 |
| registerAllCellTypes | - | - | 注册所有内置单元格类型 |
| registerAllPlugins | - | - | 注册所有内置插件 |

## 使用示例

```ts
import { registerAllModules } from 'handsontable/registry'

registerAllModules()
```
