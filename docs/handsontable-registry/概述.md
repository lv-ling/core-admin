# handsontable/registry 概述

## 基本信息

| 项 | 值 |
|----|-----|
| **包名** | handsontable/registry |
| **导入路径** | `handsontable/registry` |
| **说明** | Handsontable 模块注册工具，用于一次性注册所有内置的编辑器、渲染器、校验器、单元格类型和插件 |

## 导出

| 导出 | 类型 | 说明 |
|------|------|------|
| registerAllModules | () => void | 注册所有模块 |
| registerAllEditors | function | 注册所有内置编辑器 |
| registerAllRenderers | function | 注册所有内置渲染器 |
| registerAllValidators | function | 注册所有内置校验器 |
| registerAllCellTypes | function | 注册所有内置单元格类型 |
| registerAllPlugins | function | 注册所有内置插件 |

## 使用方式

```ts
import { registerAllModules } from 'handsontable/registry'

registerAllModules()
```

与 @handsontable/vue3 配合使用：

```vue
<script setup lang="ts">
import { HotTable } from '@handsontable/vue3'
import { registerAllModules } from 'handsontable/registry'
import 'handsontable/styles/handsontable.min.css'

registerAllModules()
</script>
```
