# handsontable/registry 类型 (Types)

## 说明

`handsontable/registry` 导出的均为函数，无独立类型定义。

## 导出类型签名

| 导出 | 类型签名 |
|------|----------|
| registerAllModules | () => void |
| registerAllEditors | 从 handsontable/editors 导出 |
| registerAllRenderers | 从 handsontable/renderers 导出 |
| registerAllValidators | 从 handsontable/validators 导出 |
| registerAllCellTypes | 从 handsontable/cellTypes 导出 |
| registerAllPlugins | 从 handsontable/plugins 导出 |
