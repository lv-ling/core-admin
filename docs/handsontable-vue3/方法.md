# @handsontable/vue3 方法 (Methods)

## 获取 Handsontable 实例

通过 `ref` 获取 HotTable 组件后，访问 `hotInstance`。

```vue
<template>
  <HotTable ref="hotRef" :data="data" />
</template>

<script setup>
const hotRef = ref(null)
const hot = hotRef.value?.hotInstance
</script>
```

## Handsontable 常用方法

| 方法名 | 参数 | 返回值 | 说明 |
|--------|------|--------|------|
| loadData | (data, source?) | void | 加载/替换全部数据 |
| updateData | (data, source?) | void | 更新数据 |
| getData | (row?, col?, row2?, col2?) | CellValue[] | 获取指定范围数据 |
| getSourceData | (row?, col?, row2?, col2?) | CellValue[][] \| RowObject[] | 获取源数据 |
| setDataAtCell | (row, col, value, source?) | void | 设置单元格数据 |
| validateCells | (callback?) | void | 触发单元格校验 |
| destroy | () | void | 销毁实例 |
| render | () | void | 强制重新渲染 |
| updateSettings | (settings, init?) | void | 更新配置 |
| undo | () | void | 撤销 |
| redo | () | void | 重做 |
