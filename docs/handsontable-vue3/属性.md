# @handsontable/vue3 属性 (Props)

## HotTable 组件

HotTable 的 props 继承自 `Handsontable.GridSettings`，并额外支持：

| 属性名 | 类型 | 说明 |
|--------|------|------|
| id | string | 表格根元素的 id |
| settings | Handsontable.GridSettings | 一次性传入的完整配置对象 |
| [additional] | any | 支持 Handsontable 所有 GridSettings 属性透传 |

## 常用 GridSettings 属性

| 属性名 | 类型 | 说明 |
|--------|------|------|
| data | CellValue[][] \| RowObject[] | 表格数据源 |
| colHeaders | boolean \| string[] \| (index) => string | 列头显示方式 |
| rowHeaders | boolean \| string[] \| (index) => string | 行头显示方式 |
| columns | ColumnSettings[] \| (index) => ColumnSettings | 列配置 |
| width | number \| string \| () => number \| string | 表格宽度 |
| height | number \| string \| () => number \| string | 表格高度 |
| licenseKey | string \| 'non-commercial-and-evaluation' | 许可证密钥 |
| themeName | string | 主题名称（如 ht-theme-main） |
| stretchH | 'none' \| 'all' \| 'last' | 列宽拉伸模式 |
| autoWrapRow | boolean | 行自动换行 |
| autoWrapCol | boolean | 列自动换行 |
| readOnly | boolean | 是否只读 |
| allowInvalid | boolean | 是否允许校验未通过的单元格 |

## HotColumn 组件

| 属性名 | 类型 | 说明 |
|--------|------|------|
| data | string \| number | 列对应的数据字段 |
| type | CellType \| string | 单元格类型（text、numeric、dropdown 等） |
| readOnly | boolean | 列是否只读 |
| width | number | 列宽 |
| editor | EditorType | 编辑器类型 |
| renderer | RendererType | 渲染器类型 |
| validator | ValidatorType | 校验器 |
| source | string[] \| (query, callback) => void | dropdown 选项源 |
