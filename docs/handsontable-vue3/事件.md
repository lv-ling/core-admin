# @handsontable/vue3 事件 (Events)

通过 HotTable 的 props 传入 Handsontable Hooks 作为回调。

## 数据变更

| 事件名 | 参数 | 说明 |
|--------|------|------|
| afterChange | (changes, source) => void | 单元格数据变更后触发 |
| beforeChange | (changes, source) => boolean \| void | 变更前，返回 false 可阻止 |

## 选择与焦点

| 事件名 | 参数 | 说明 |
|--------|------|------|
| afterSelection | (row, column, row2, column2, ...) => void | 选择变更后 |
| afterDeselect | () => void | 取消选择后 |

## 编辑

| 事件名 | 参数 | 说明 |
|--------|------|------|
| afterBeginEditing | (row, column) => void | 开始编辑后 |
| beforeBeginEditing | (...) => boolean \| void | 开始编辑前 |

## 校验

| 事件名 | 参数 | 说明 |
|--------|------|------|
| afterValidate | (isValid, value, row, prop, source) => void \| boolean | 单元格校验后 |

## 渲染与初始化

| 事件名 | 参数 | 说明 |
|--------|------|------|
| afterInit | () => void | 表格初始化完成后 |
| afterRender | (isForced: boolean) => void | 渲染完成后 |
| afterUpdateSettings | (newSettings) => void | 配置更新后 |

## 加载数据

| 事件名 | 参数 | 说明 |
|--------|------|------|
| afterLoadData | (sourceData, initialLoad, source) => void | 数据加载后 |
| afterUpdateData | (sourceData, initialLoad, source) => void | 数据更新后 |
