# @handsontable/vue3 类型 (Types)

## 导出的主要类型

| 类型名 | 说明 |
|--------|------|
| HotTable | Vue 3 表格组件 |
| HotColumn | Vue 3 列定义组件 |

## HotTableProps

```ts
interface HotTableProps extends Handsontable.GridSettings {
  id?: string
  settings?: Handsontable.GridSettings
  [additional: string]: any
}
```

## VueProps<T>

```ts
type VueProps<T> = { [P in keyof T]: any }
```

## Handsontable 相关类型

| 类型 | 说明 |
|------|------|
| GridSettings | 表格配置接口 |
| ColumnSettings | 列配置接口 |
| CellMeta | 单元格元数据 |
| CellProperties | 渲染后的单元格对象 |
| CellValue | 单元格值类型 |
| RowObject | 行对象 |
| ChangeSource | 变更来源 |
| CellChange | 单元格变更描述 |
