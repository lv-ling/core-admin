# @handsontable/vue3 概述

## 基本信息

| 项 | 值 |
|----|-----|
| **包名** | @handsontable/vue3 |
| **版本** | 16.2.0 |
| **描述** | 为 Vue 3 提供具有电子表格外观与交互的 Handsontable 数据表格封装 |
| **主页** | https://handsontable.com |
| **仓库** | https://github.com/handsontable/handsontable |

## 依赖

| 类型 | 包名 | 版本 |
|------|------|------|
| peer | handsontable | >=16.0.0 |
| peer | vue | ^3.2.22 |

## 导出

| 导出 | 说明 |
|------|------|
| HotTable | 主表格组件，对应 Handsontable 实例 |
| HotColumn | 列定义组件，用于声明式定义列配置 |
| default | 同 HotTable |

## 入口

| 类型 | 路径 |
|------|------|
| main (CJS) | ./commonjs/vue-handsontable.js |
| module (ESM) | ./es/vue-handsontable.mjs |
| types | ./index.d.ts |

## 子路径导出

- `.`：主入口
- `./dist/vue-handsontable.js`
- `./dist/vue-handsontable.min.js`

## 使用示例

```vue
<script setup lang="ts">
import { HotTable } from '@handsontable/vue3'
import { registerAllModules } from 'handsontable/registry'
import 'handsontable/styles/handsontable.min.css'
import 'handsontable/styles/ht-theme-main.min.css'

registerAllModules()
</script>

<template>
  <HotTable
    ref="hotRef"
    :data="data"
    :col-headers="true"
    :row-headers="true"
    license-key="non-commercial-and-evaluation"
  />
</template>
```
