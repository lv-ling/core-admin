# ag-grid-community 属性 (Props)

ag-grid-community 是核心库，配置通过 **GridOptions** 传入 AgGridVue。

## GridOptions 核心属性

| 属性名 | 类型 | 说明 |
|--------|------|------|
| columnDefs | (ColDef \| ColGroupDef)[] | 列定义 |
| defaultColDef | ColDef | 默认列定义 |
| rowData | TData[] | 行数据 |
| getRowId | GetRowIdFunc | 行 ID 获取函数 |
| context | any | 上下文 |

## 列相关

| 属性名 | 类型 | 说明 |
|--------|------|------|
| headerHeight | number | 表头高度 |
| suppressMovableColumns | boolean | 是否禁止列拖拽 |
| suppressColumnMoveAnimation | boolean | 是否关闭列移动动画 |

## 行相关

| 属性名 | 类型 | 说明 |
|--------|------|------|
| rowHeight | number | 行高 |
| rowSelection | 'single' \| 'multiple' \| RowSelectionOptions | 行选择模式 |
| rowMultiSelectWithClick | boolean | 点击多选 |
| suppressRowClickSelection | boolean | 是否禁止点击选中行 |

## 编辑相关

| 属性名 | 类型 | 说明 |
|--------|------|------|
| singleClickEdit | boolean | 单击即编辑 |
| stopEditingWhenCellsLoseFocus | boolean | 失焦时停止编辑 |
| invalidEditValueMode | 'revert' \| 'block' | 校验失败时的行为 |
