# ag-grid-community 类型 (Types)

## 核心类型

| 类型名 | 说明 |
|--------|------|
| GridApi | 表格 API 实例 |
| GridOptions\<TData\> | 表格配置接口 |
| ColDef | 列定义 |
| ColGroupDef | 列组定义 |
| IRowNode | 行节点 |
| Module | AG Grid 模块 |

## 事件类型

| 类型名 | 说明 |
|--------|------|
| GridReadyEvent | grid-ready 事件参数 |
| CellValueChangedEvent | 单元格值变化事件 |
| SelectionChangedEvent | 选择变更事件 |
| FilterChangedEvent | 筛选变化事件 |

## 其他常用类型

| 类型名 | 说明 |
|--------|------|
| GetRowIdFunc | 行 ID 获取函数 |
| StatusPanelDef | 状态栏面板定义 |
| SideBarDef | 侧边栏定义 |
| RowSelectionOptions | 行选择选项 |
