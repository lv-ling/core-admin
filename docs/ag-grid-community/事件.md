# ag-grid-community 事件 (Events)

事件通过 GridApi 的 `addEventListener` 添加，或在 AgGridVue 上使用 `@事件名` 绑定（kebab-case）。

## 常用事件类型

| 事件名 | 说明 |
|--------|------|
| gridReady | 表格初始化完成 |
| cellValueChanged | 单元格值变化 |
| cellClicked | 单元格点击 |
| cellDoubleClicked | 单元格双击 |
| selectionChanged | 选择变更 |
| filterChanged | 筛选变化 |
| columnResized | 列宽调整 |
| columnMoved | 列移动 |
| columnVisible | 列显示/隐藏 |
| columnPinned | 列固定/取消固定 |
| rowSelected | 行选中/取消选中 |
| cellEditingStarted | 单元格编辑开始 |
| cellEditingStopped | 单元格编辑结束 |

## 添加事件监听

```ts
api.addEventListener('cellValueChanged', (event) => {
  console.log('cell changed', event.data)
})
```

## 移除监听

```ts
const handler = (e) => { ... }
api.addEventListener('cellValueChanged', handler)
api.removeEventListener('cellValueChanged', handler)
```
