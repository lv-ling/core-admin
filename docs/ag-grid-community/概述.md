# ag-grid-community 概述

## 基本信息

| 项 | 值 |
|----|-----|
| **包名** | ag-grid-community |
| **版本** | 35.0.1 |
| **描述** | 高级数据表格，支持 JavaScript/TypeScript/React/Angular/Vue |
| **主页** | https://www.ag-grid.com/ |
| **仓库** | https://github.com/ag-grid/ag-grid |

## 依赖

| 类型 | 包名 | 版本 |
|------|------|------|
| dependency | ag-charts-types | 13.0.1 |

## 主要导出

- **GridApi**：表格 API
- **GridOptions**：表格配置
- **ColDef**、**ColGroupDef**：列定义
- **IRowNode**：行节点
- **Module**：模块化导入
- **各类 Event 类型**：事件参数类型

## 入口

| 类型 | 路径 |
|------|------|
| main (CJS) | ./dist/package/main.cjs.js |
| module (ESM) | ./dist/package/main.esm.mjs |
| types | ./dist/types/src/main.d.ts |

## 样式导出

- `ag-grid.css` / `ag-grid.min.css`
- `ag-theme-quartz.css`、`ag-theme-alpine.css`、`ag-theme-balham.css`、`ag-theme-material.css` 等

## 使用方式

```ts
import 'ag-grid-community/styles/ag-grid.css'
import 'ag-grid-community/styles/ag-theme-quartz.css'
import { AgGridVue } from 'ag-grid-vue3'
import type { GridOptions } from 'ag-grid-community'
```
